## Dependências necessárias

- `pip install pypdf`

- `pip install nltk`

- `pip install rank_bm25`

- `pip install fastapi`

- `pip install uvcorn`

## Rodar backend

Na raiz do diretório executar o comando: `uvicorn api:app --reload`

## Rodar frontend

Acessar o arquivo `index.html` pelo navegador.
